# 📚 Java Programming & DSA Practice

A collection of Java programs covering **core Java fundamentals** and **Data Structures & Algorithms (DSA)** concepts, written while learning the Java programming language from scratch.

---

## 👩‍💻 About

This repository contains hands-on practice programs written in Java as part of a self-learning journey. It covers everything from basic syntax and operators all the way up to classic DSA problems like sorting algorithms, matrix traversal, and subarray problems.

## 🧠 Key Algorithms Implemented

- **Bubble Sort** — Comparison-based sorting with nested loops
- **Kadane's Algorithm** — Efficient O(n) maximum subarray sum
- **Spiral Matrix Traversal** — Layer-by-layer matrix printing
- **Diagonal Sum** — Primary + secondary diagonal sum avoiding double-count
- **Shortest Path** — Coordinate-based distance using the Euclidean formula
- **Binary to Decimal** — Manual conversion using positional values
- **Bitwise Operations** — Get bit, set bit, range extraction

## 🛠️ How to Run

Make sure you have **Java JDK** installed. Then compile and run any file:

```bash
# Compile
javac FileName.java

# Run
java FileName
```

**Example:**
```bash
javac bubbleSort.java
java bubbleSort
```
